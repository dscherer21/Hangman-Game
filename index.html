Hangman

Instructions

Press any key to start!

  • Type a letter and if it is part of the word or phrase, it will show inside the blanks for each instance of that letter.

  • If you make a wrong guess, it will be logged in the guessed letters box.

  • You have 8 wrong guesses before you get a game loss.

  • After you win or lose, the computer will reset to a new word.
